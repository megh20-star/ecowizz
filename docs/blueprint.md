# **App Name**: EcoHabit Tracker

## Core Features:

- Smart Reminders: Set smart reminders using AI to switch off appliances and reduce wastage, analyzing user behavior to predict when reminders are most effective as a tool.
- Usage Tracking: Track daily energy and water usage with visual charts and graphs for improved consumption control.
- Saving Tips: Provide practical, tailored tips for saving resources and adopting sustainable habits.
- Competition Mode: Enable schools/colleges to run 'Save Energy/Water' competitions, tracking progress on a leaderboard. Save users usage information to the database.
- Profile and Settings: Allow users to set up a personal profile, manage notification preferences and view environmental stats.

## Style Guidelines:

- Primary color: Deep green (#2E8B57), reminiscent of nature and growth.
- Background color: Soft, desaturated green (#E0F2E9) for a calming and eco-friendly feel.
- Accent color: Muted teal (#3CB371) to highlight key interactive elements and calls to action.
- Font: 'PT Sans', a humanist sans-serif providing both a modern look with warmth and personality, used for both headlines and body text.
- Use clean, simple icons to represent different appliances, resources, and actions.
- Clean and intuitive interface layout with a focus on smooth user interaction.
- Subtle animations to provide feedback and improve user engagement.
